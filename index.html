O que é HTML?

O HTML é uma Linguagem de Marcação utilizada na criação de páginas WEB, desenvolvida originalmente pelo físico britânico Tim Berners-Lee. Todo documento HTML é constituído por marcadores (em inglês tags): palavras entre os sinais menor que (<) e maior que (>). Os marcadores podem ter atributos, valores e/ou filhos e podem ser utilizados sozinhos ou em conjunto com o marcador de fechamento, em que há a adição do caractere “/”. Por exemplo:

Marcador com abertura e fechamento:
<p>Nam nec odio vel turpis.</p> (Adiciona um parágrafo)

Marcador apenas com abertura:
<br>     (Realiza quebra de linha)

Todo elemento HTML representa um item da página, como um parágrafo, uma imagem, um link, um formulário, etc. Existem dois tipo de elementos:

  1. Elementos de bloco:

    2. São isolados pela adição de uma linha antes e após a exibição do conteúdo e ocupam toda a largura disponível.

  2. Elementos inline:

    3. Ocupam apenas o espaço correspondente ao que será inserido na tela.

Uso do HTML

Como já vimos anteriormente, o HTML não é uma linguagem de programação mas sim uma linguagem de marcação que contém uma série de tags que possibilitam a criação de conteúdos com diferentes funcionalidades.

O HTML está principalmente na parte visual das aplicações, na parte do frontend. Em conjunto com o CSS e o Javascript, formam a base do frontend e das páginas web. Em outras palavras, Frontend é a parte da aplicação que interage diretamente com o usuário.

Estes são alguns dos usos mais comuns para o HTML:

  • Desenvolvimento Web

    Os desenvolvedores usam códigos HTML para projetar como um navegador vai exibir os elementos das páginas, como textos, formulários, hiperlinks e arquivos de mídia.

  • Navegação na internet

    Os usuários podem navegar facilmente e inserir links entre páginas e sites relacionados, já que o HTML é amplamente usado para incorporar hiperlinks.

  • Documentação

    O HTML torna possível a organização e a formatação de documentos.
